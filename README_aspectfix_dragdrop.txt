画像アスペクト補正 GUI
======================

概要
----
既存画像の後処理用 GUI です。
フレーム抽出、アップスケール、画像生成は行いません。

主機能は、横倍率と縦倍率を使って画像内容そのものを実際に変形することです。

    new_width  = round(width  * XScale)
    new_height = round(height * YScale)

元画像は上書きせず、必ず `OutputFolder` 配下へ別名保存します。


初期値
------
初期値は `720x480` の格納比率画像を `16:9` 表示相当に近づけるための値です。

- 横倍率: `1.185185`
- 縦倍率: `1.000000`
- 初期プリセット: `720x480 → 16:9表示相当補正`
- 初期キャンバス処理: `なし`

この値は固定ではなく、GUI 上で自由に変更できます。
まず数枚を初期値でプレビューし、自然に見える値へ微調整してください。


重要
----
- 主機能は横倍率・縦倍率による画像内容そのものの補正です。
- `指定比率キャンバスに配置` や `ぼかし背景` は保存時の補助機能で、人物比率を直す機能ではありません。
- 元画像は絶対に上書きしません。


基本操作
--------
1. `run_aspectfix_dragdrop.bat` をダブルクリックして起動します。
2. エクスプローラーで画像ファイルを複数選択します。
3. GUI のドロップ領域へドラッグ＆ドロップします。
4. 必要なら `ファイル追加` または `フォルダから追加` を使います。
5. `倍率補正` で横倍率 / 縦倍率、またはプリセットを調整します。
6. 初期状態の `キャンバス処理: なし` のまま `プレビュー更新` を押します。
7. 右側の別ウィンドウで倍率補正後の画像内容を確認します。
8. 必要なら補助機能としてキャンバス処理を追加します。
9. `処理開始` を押してリスト内画像だけを一括処理します。


プリセット
----------
- `720x480 → 16:9表示相当補正`
  - 横倍率: `1.185185`
  - 縦倍率: `1.000000`
- `変更なし`
  - 横倍率: `1.000000`
  - 縦倍率: `1.000000`
- `横を少し縮める`
  - 横倍率: `0.980000`
  - 縦倍率: `1.000000`
- `横を縮める`
  - 横倍率: `0.950000`
  - 縦倍率: `1.000000`
- `横を少し広げる`
  - 横倍率: `1.050000`
  - 縦倍率: `1.000000`
- `縦を少し縮める`
  - 横倍率: `1.000000`
  - 縦倍率: `0.980000`
- `縦を縮める`
  - 横倍率: `1.000000`
  - 縦倍率: `0.950000`
- `カスタム`
  - 現在の入力欄の値を使います


保存時のキャンバス処理
----------------------
- `なし`
- `元画像サイズに中央配置`
- `元画像サイズに中央クロップ`
- `指定比率キャンバスに配置`
- `ぼかし背景`

初期状態は `なし` です。
先に倍率補正だけで自然に見えるかを確認してください。


プレビュー
----------
- 左: 補正前
- 右: 倍率補正後の画像内容
- 別ウィンドウの状態表示
  - 元サイズ
  - 倍率補正後サイズ
  - 横倍率
  - 縦倍率
  - 元比率
  - 倍率補正後比率
  - キャンバス処理の有無

初期状態では `倍率補正のみ / キャンバス処理なし` と表示されます。


保存処理順
----------
1. 元画像を読み込む
2. 横倍率・縦倍率で画像内容をリサイズする
3. キャンバス処理が有効なら、その後にキャンバス処理する
4. 保存する


出力仕様
--------
- 元画像は絶対に上書きしません
- 出力名は元ファイル名に接尾辞を付けます
- 同名が存在する場合は `_001`, `_002` のように連番回避します

例:
- `input.png` -> `input_aspectfix.png`


設定ファイル
------------
初期 `aspectfix_dragdrop_config.json` の重要項目は次のとおりです。

```json
{
  "XScale": 1.185185,
  "YScale": 1.0,
  "DefaultPreset": "720x480 → 16:9表示相当補正",
  "CanvasMode": "なし",
  "Suffix": "_aspectfix",
  "Overwrite": false
}
```

実際の設定ファイルには、出力フォルダ、対象拡張子、画質、ウィンドウサイズなどの補助設定も保存されます。


ログとレポート
--------------
GUI ログ、`OutputFolder\\logs\\` のログ、`aspectfix_report_日時.txt` はすべて日本語です。

記録内容:
- 入力ファイル
- 元サイズ
- 横倍率
- 縦倍率
- 倍率補正後サイズ
- キャンバス処理
- 出力ファイル
- 成功 / 失敗
- エラー内容


セットアップ
------------
```powershell
py -3 -m pip install -r requirements_aspectfix.txt
```


起動
----
- 推奨: `run_aspectfix_dragdrop.bat`
- 直接起動: `py -3 aspectfix_dragdrop_gui.py`
- GUI 生成確認: `py -3 aspectfix_dragdrop_gui.py --headless-smoke-test`
